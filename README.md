# EduTrack Pro

EduTrack Pro, eğitim süreçlerinizi daha etkin bir şekilde yönetmenizi sağlayan bir uygulamadır.

## Özellikler
- Kullanıcı dostu arayüz
- Eğitim planları oluşturma
- Görev atama ve izleme
- Raporlama ve analiz araçları

## Kurulum
1. Projeyi klonlayın:
   ```bash
   git clone https://github.com/yogunes/edutrack-pro.git
   ```
2. Gerekli paketleri yükleyin:
   ```bash
   npm install
   ```
3. Uygulamayı başlatın:
   ```bash
   npm start
   ```

## Kullanım
Eğitim süreçlerinizi yönetmek için uygulamayı açın ve ihtiyaçlarınıza göre özelleştirin.
